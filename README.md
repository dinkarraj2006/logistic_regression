## 🎯 Project Overview

This project is a complete web-based educational tool that demonstrates Logistic Regression from first principles. It allows users to:

- Upload their own CSV datasets
- Preprocess data (handling missing values, encoding, scaling)
- Perform Exploratory Data Analysis (EDA)
- Understand the mathematical foundations of Logistic Regression
- Train a model using Gradient Descent (implemented from scratch)
- Evaluate model performance with multiple metrics
- Make predictions on new data

**No external ML libraries were used** - all algorithms are implemented in pure JavaScript.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📁 **Dataset Upload** | Upload CSV files with binary/multi-class target variables |
| 🔧 **Preprocessing** | Missing value imputation, label encoding, standardization |
| 📊 **EDA** | Class distribution, feature distributions, correlation matrix |
| 📐 **Mathematical Theory** | Sigmoid function, probability interpretation, decision boundary |
| 🚀 **Training from Scratch** | Gradient descent implementation (no ML libraries) |
| 📈 **Evaluation** | Confusion matrix, accuracy, precision, recall, F1-score, ROC curve |
| 🔮 **Prediction** | Custom sample inference with step-by-step calculation |
| 🔄 **Cross-Validation** | K-Fold cross-validation support |
