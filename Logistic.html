Logistic Regression: Mathematical Intuition

📐 Linear Model → Sigmoid
z = w·x + b → σ(z) = 1 / (1 + e^{-z})

Probability P(y=1|x) = σ(z). Decision boundary at σ(z)=0.5 => z=0.

Why Sigmoid? Maps any real number to (0,1) allowing probability interpretation.

🔄 Binary vs Multi-class (OvR)

Binary: single sigmoid. Multi-class: One-vs-Rest trains K classifiers, picks highest probability.
argmax(σ(z_k))

📉 Linear vs Logistic Regression

Linear => continuous output; Logistic => probability + classification threshold. Loss: Cross-Entropy instead of MSE.
